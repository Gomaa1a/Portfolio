🔴 Redis State Management
Integrated Redis DB for message buffering — solving the race condition where multiple messages arrive before the agent responds. Ensures zero message loss under high volume.
🔌 4 Live API Connections
Agent connects in real-time to: availability checker, pricing engine, user history lookup by phone number (not name — solving duplicate identity problem), and booking confirmation endpoint.
🎯 Sentiment Detection Layer
A separate intent pipeline inside the main agent detects compliment messages and automatically routes them to the marketing team — turning positive customer feedback into a marketing asset.
🧠 Conversation Memory
Agent maintains full conversation context across turns. Each session is stateful — the agent knows what was discussed, what was quoted, and what actions were already taken.
WORKFLOW 01 / 04
Main AI Booking Agent
Central brain of the system. Redis-buffered. 4 live API connections. Sentiment detection. Conversation memory. Routes leads to sub-workflow. Handles full booking flow autonomously.
WORKFLOW 02 / 04
OCR Payment Verifier
Customer sends payment screenshot. OCR extracts amount and date. System validates against expected payment. Returns verified true/false directly to website backend. No manual checking ever.
WORKFLOW 03 / 04
Lead Processing Sub-Workflow
Triggered by main agent on lead detection. Reformats raw lead data into clean structured format. Writes to database. Sends formatted lead card to team WhatsApp instantly.
WORKFLOW 04 / 04
Booking Confirmation Pipeline
After completed booking, captures the full booking record. Writes to database. Pushes structured booking data to team. Closes the loop on the entire customer journey.
